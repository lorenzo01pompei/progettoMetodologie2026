# 📌 RPG Worlds

**RPG Worlds** è un videogioco di ruolo che permette al giocatore di:

* esplorare diversi mondi, ognuno con i propri nemici da affrontare;
* aggiungere equipaggiamento all'inventario;
* recuperare materiali dalle miniere dei mondi e scambiarli per monete.

---

# 🚀 Come eseguire il progetto

## Prerequisiti

* Java 25 (LTS)
* Gradle

## Istruzioni

```bash
git clone https://github.com/lorenzo01pompei/progettoMetodologie2026
cd progettoMetodologie
```

## Build del progetto

```bash
./gradlew build
```

## Esecuzione

```bash
./gradlew run
```

# 🤖 Uso di strumenti di AI

Per lo sviluppo di **RPG Worlds** è stato utilizzato **Gemini (Google AI)** come assistente alla programmazione.

L'intelligenza artificiale è stata impiegata come supporto tecnico per scopi di conoscenza e risoluzione di errori.

Tutta la struttura principale del progetto, le dinamiche di gioco e il flusso dell'applicazione sono stati progettati da me, utilizzando l'AI soltanto come strumento di facilitazione.

## Attività svolte con il supporto dell'AI

### 📚 Supporto e approfondimento teorico

Utilizzo dell'AI per:

* comprendere meglio i concetti teorici;
* applicare design pattern;
* analizzare soluzioni ottimali in Java.

### 🗄️ Precompilazione del database

Supporto nella generazione di:

* dati iniziali;
* script per la precompilazione del database;

per permettere il testing del gioco e dei suoi elementi (nemici, oggetti, ecc.).

### 📝 Realizzazione di commenti

Supporto nella scrittura di:

* documentazione;
* commenti del codice.

Tutti i contenuti sono stati comunque letti, revisionati e adattati personalmente.

### 🐞 Debugging e risoluzione errori

Aiuto nell'analisi e nella correzione di:

* bug;
* errori di compilazione;
* problemi emersi durante integrazione e testing del sistema.
